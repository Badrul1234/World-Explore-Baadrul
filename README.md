# üåç World Explore

World Explore is a **desktop application** built using **Electron.js**, designed for students and explorers to discover information about countries across the world in a simple and interactive way.

---

##  Features
-  Search and view detailed country data in real-time  
-  Display flag, population, region, capital, and languages  
-  Save favorite countries to a personal watchlist  
-  Edit or delete entries (Full CRUD operations)  
-  Interactive user interface with HTML, CSS, and JavaScript  

---

##  External API
**REST Countries API**  
Used to fetch live country data such as:
- Name  
- Capital  
- Population  
- Region  
- Flag  
- Languages  

API Endpoint Example:
```js
fetch('https://restcountries.com/v3.1/name/malaysia')
